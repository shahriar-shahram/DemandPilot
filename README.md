# DemandPilot

An end-to-end demand forecasting and analytics system built from real retail transaction data.

## Features

- Data cleaning and transformation pipeline
- Time-series aggregation of retail demand
- Feature engineering with business signals (discount, volume, product diversity)
- Interactive dashboard with filtering (Streamlit)
- Region-level demand forecasting models
- Evaluation using MAE, RMSE, and SMAPE

## Tech Stack

- Python
- Pandas
- Streamlit
- Scikit-learn
- Matplotlib

## How to Run

```bash
pip install -r requirements.txt
streamlit run app/main.py
