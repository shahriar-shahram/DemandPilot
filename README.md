# DemandPilot

An AI-driven demand forecasting system under development.

## Current Stage
- Data pipeline setup
- Dataset integration (Walmart sales data)

## Current Features

- Retail transaction data cleaning pipeline
- Daily sales aggregation for demand analysis
- Interactive dashboard with:
  - KPI cards
  - sales trends
  - monthly trends
  - region and category filters
  - profit analysis
  - top product table
